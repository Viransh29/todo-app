#  To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS (Tailwind), and JavaScript.  
It allows users to add, delete, and manage daily tasks efficiently.

---

##  Features

-  Add new tasks
-  Delete tasks
-  Saves tasks in Local Storage (data persists after refresh)

---

##  Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- Local Storage API

---

##  How to Run the Project
